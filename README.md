📒 Quản Lý Danh Bạ - Python
Ứng dụng quản lý danh bạ chạy trên terminal — thêm, xóa, sửa, tìm kiếm và thống kê liên hệ.

✨ Tính năng
👁️ Xem toàn bộ danh bạ
➕ Thêm liên hệ mới (tên, SĐT, email, nhóm)
🔍 Tìm kiếm theo tên (không phân biệt hoa/thường)
❌ Xóa liên hệ
✏️ Sửa số điện thoại
📊 Thống kê tổng số liên hệ và theo nhóm

🎬 Demo
Hãy lựa chọn phương án của bạn
    1. Xem toàn bộ danh bạ
    2. Thêm liên hệ
    3. Tìm kiếm theo tên
    4. Xóa liên hệ
    5. Sửa số điện thoại
    6. Thống kê
    7. Thoát
Nhập lựa chọn của bạn: 6
Tổng số liên hệ là 5
Thống kê theo nhóm:
 Gia đình: 2 liên hệ
 Bạn bè: 2 liên hệ
 Công việc: 1 liên hệ

 🛠️ Kiến thức áp dụng
List - lưu toàn bộ danh bạ
Dict - lưu thông tin từng liên hệ
Set - lấy danh sách nhóm duy nhất
while True + break - vòng lặp menu
